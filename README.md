# 🏠 Erste Wohnung – Survival Guide

**Ein digitaler Begleiter für den Start in die erste eigene Wohnung**  
Entwickelt als Begleitmaterial für den WP10-Unterricht – aber frei nutzbar für alle.

> Stand: März 2026

---

## 📖 Was ist das?

Diese Website ist ein alltagsnaher Nachschlage- und Inspirationsraum rund um das Thema „erste eigene Wohnung". Sie richtet sich an Jugendliche und junge Erwachsene, die sich auf den Auszug vorbereiten oder einfach wissen wollen, wie das alles funktioniert.

Die Seite ist **kein Arbeitsheft und keine Pflichtlektüre** – sondern ein digitaler Begleiter, der schnelle Orientierung, ehrliche Tipps und klare Hinweise gibt.

---

## 📂 Inhalt

Die Website besteht aus 7 vollständig verlinkten HTML-Seiten:

| Datei | Inhalt |
|---|---|
| `index.html` | Startseite mit Kacheln, FAQ und Symbolsystem |
| `wohnen.html` | Wohnen & Alltag – Einrichten, Ordnung, Grundausstattung |
| `technik.html` | Technik & Reparatur – Bohren, Dübel, Werkzeuge, Materialien |
| `strom-wasser.html` | Strom, Wasser & Heizung – Notfälle, Sparen, Lüften, Schimmel |
| `geld-vertrag.html` | Geld & Vertrag – Kosten, Mietvertrag, Budget, WG-Themen |
| `ki-hilfe.html` | Digitale Hilfe & KI – Wann KI hilft, Datenschutz, Sicherheit |
| `links.html` | Nützliche Links – sortiert nach Thema, mit Unterrichtsaufgabe |

---

## ✨ Features

- **Vollständig statisch** – kein Server, kein Framework, keine Abhängigkeiten
- **Single-File-Seiten** – HTML, CSS und JavaScript jeweils in einer Datei
- **Responsiv** – funktioniert auf Handy, iPad und Desktop
- **Interaktive Elemente** – Checklisten mit `localStorage`, Kostenrechner, Tabs, Akkordeons
- **Einheitliches Designsystem** – jede Seite hat eine eigene Akzentfarbe, gleiches Layout
- **Barrierearm** – ausreichende Kontrastverhältnisse, Touch-optimierte Bedienelemente
- **Datenschutzhinweise** – expliziter Umgang mit dem Thema KI und persönliche Daten

---

## ⚠️ Wichtige Hinweise

Diese Website enthält explizite Sicherheitshinweise:

- Bestimmte Arbeiten (Elektro, Gas, Wasser, tragende Wände) **dürfen nur von Fachkräften** durchgeführt werden – unabhängig von Anleitungen im Internet oder KI-Empfehlungen.
- KI-Antworten müssen **immer an einer zweiten Quelle geprüft** werden.
- In KI-Tools sollten **keine persönlichen Daten** eingegeben werden.
- Die Inhalte haben den **Stand März 2026**. Besonders im KI-Bereich können sich Informationen schnell ändern.

---

## 🚀 Deployment

Die Website kann direkt über [Vercel](https://vercel.com) veröffentlicht werden:

1. Repository auf GitHub anlegen und alle 7 HTML-Dateien hochladen
2. Auf vercel.com mit GitHub-Account anmelden
3. Repository importieren – **keine Konfiguration nötig**
4. „Deploy" klicken → fertig

Vercel erkennt automatisch, dass es sich um statische HTML-Dateien handelt. Änderungen im GitHub-Repository werden automatisch deployed.

# 🏠 First Apartment – Survival Guide

**A digital companion for moving into your first own apartment**  
Developed as accompanying material for WP10 classes – but free to use for everyone.

> As of: March 2026

---

## 📖 What is this?

This website is a practical reference and inspiration resource about moving into your first apartment. It is aimed at teenagers and young adults preparing to move out, or anyone who wants to understand how it all works.

The site is **not a workbook and not required reading** – it's a digital companion that provides quick orientation, honest tips, and clear guidance.

---

## 📂 Contents

The website consists of 7 fully linked HTML pages:

| File | Content |
|---|---|
| `index.html` | Home page with topic tiles, FAQ and symbol system |
| `wohnen.html` | Living & Everyday Life – furnishing, tidiness, basic equipment |
| `technik.html` | Tech & Repairs – drilling, wall plugs, tools, materials |
| `strom-wasser.html` | Electricity, Water & Heating – emergencies, saving energy, ventilation, mould |
| `geld-vertrag.html` | Money & Contracts – costs, rental agreements, budget, flat-share topics |
| `ki-hilfe.html` | Digital Help & AI – when AI helps, data protection, safety |
| `links.html` | Useful Links – sorted by topic, with a classroom activity |

---

## ✨ Features

- **Fully static** – no server, no framework, no dependencies
- **Single-file pages** – HTML, CSS and JavaScript in one file each
- **Responsive** – works on phone, iPad and desktop
- **Interactive elements** – checklists with `localStorage`, cost calculator, tabs, accordions
- **Consistent design system** – each page has its own accent colour, same layout
- **Accessible** – sufficient contrast ratios, touch-optimised controls
- **Data privacy guidance** – explicit handling of AI tools and personal data

---

## ⚠️ Important Notes

This website contains explicit safety warnings:

- Certain tasks (electrical work, gas, water pipes, load-bearing walls) **must only be carried out by qualified professionals** – regardless of online guides or AI recommendations.
- AI responses must **always be verified against a second source**.
- **No personal data** should be entered into AI tools.
- The content reflects the status of **March 2026**. Especially in the area of AI, information can change rapidly.

---

## 🚀 Deployment

The website can be published directly via [Vercel](https://vercel.com):

1. Create a repository on GitHub and upload all 7 HTML files
2. Sign in to vercel.com with your GitHub account
3. Import the repository – **no configuration needed**
4. Click "Deploy" → done

Vercel automatically detects that these are static HTML files. Changes pushed to the GitHub repository are deployed automatically.

---

## 🎓 Educational Context

The website accompanies the final project in **WP10 classes** (elective subject, Year 10). It is not the focus of assessment, but serves as:

- A **reference resource** for everyday topics
- An **inspiration space** for personal projects
- A **conversation starter** in class (every page includes a discussion prompt)
- A **media literacy component** – especially the "Digital Help & AI" page
